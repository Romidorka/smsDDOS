# smsDDOS

smsDDOS - это программа для спамминга.

## Установка

Для Termux:

```bash
pkg install python
pip install requests argparse colorama
cd ~
git clone https://github.com/Romidorka/smsDDOS
```


## Запуск

Для Termux:

```bash
cd ~/smsDDOS
python3 main.py --phone {название файла с номерами телефонов}
```
